# lab-appindex-hth

## 项目简介

本项目是一个用于归档和发布多个独立 HTML 页面的仓库。  
这里存放的页面通常用于快速索引、工具展示或小型应用界面，不针对任何特定域名或具体网站。  
所有页面均为纯前端实现，可直接在浏览器中打开使用。

## 目录说明

```
lab-appindex-hth/
├── index.html          # 主入口页面，用于导航至各个子页面
├── pages/              # 存放独立 HTML 页面的目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/             # 公共资源文件（CSS、JS、图片等）
    ├── style.css
    └── script.js
```

- `index.html`：仓库首页，提供页面列表与快速跳转链接。
- `pages/`：每个独立 HTML 页面放置在此目录下，便于分类管理。
- `assets/`：存放多个页面共用的样式、脚本及静态资源。

## 页面归档说明

- 页面内容以实用工具、演示或索引为主，不涉及商业或特定平台功能。
- 每个页面均为自包含的 HTML 文件，可独立运行。
- 归档过程中已对页面进行基础整理，确保无外部依赖异常或安全风险。
